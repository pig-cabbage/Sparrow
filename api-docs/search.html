Sparrow Search

APIs for Sparrow File Searching

Data Structure

MSON(Markdown Syntax for Object Notation)是blueprint api使用的对象描述语言，用于抽象地描述JSON, XML, YAML等数据传输格式。

用MSON格式定义API的数据结构，用于：

  • Request Parameters

  • Request Body

  • Response Body

result - API返回数据格式（Response Body）

  • status: 200 (number) - 状态码

  • msg: OK (string) - 消息

  • data (enum, optional, nullable) - 返回数据

    • (object)
    • (array)

search_type(enum[string]) - 文件类型

  • all - 全部

  • image - 图片

  • doc - 文档

  • video - 视频

  • audio - 音频

  • other - 其它文件

S1.Search Suggestions

GET

http://localhost:8080/search/suggestions
Parameters
type type all

keyword keyword 算法

默认为空，返回默认搜索建议
size size 10

搜索建议条数
